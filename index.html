Welcome to Github

What is Github?
Github is a code hosting platform for version control and collaboration.

Why is Github used?
It lets you and others work together on projects from anywhere.

Why do I need Github?
It's used for storing, tracking, and collaborating on software projects.

Is Github good for beginners?
Github is easy to use for beginners. In fact you will only need to know a few git
commands to learn how to push code to Github.
